Calculator app made for Web Applications

This calculator app uses the Spring framework for java to create a single page applications that performs one of 5 mathematical functions
on 2 given integer numbers. The mathematical functions that can be used are:
add, subtract, multiply, divide and power (+,-,x,/,^)

-Calculator app only works for BigInteger values and will return with a 400 error if requested.
-App also creates a hash of the values given using 1 of four given possible hashing algorithms (SHA1, SHA256, SHA512, MD5)
